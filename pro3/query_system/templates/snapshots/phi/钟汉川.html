钟汉川

基本信息：

姓名： 钟汉川

性别：男

职称： 副教授

职务：

工作地址：南开大学哲学院

邮编：300350


电子邮件：zhonghanchuan@nankai.edu.cn

接待时间：周二中午一点到两点

 


个人简历：

  1. 1997年9月─2007年6月在南开大学哲学系学习获哲学学士、硕士和博士。

  2. 2007年6月—2011年12月任南开大学哲学系讲师。

  3. 2011年12月31日起至今任南开大学哲学院副教授。

  4. 2014年7月—2015年7月美国福特汉姆大学访问学者（国家留学基金委资助）。

  5. 2015年11月至今任南开大学哲学院伦理学教研室主任。


   开设过的主要课程：

   “西方哲学”、“近代欧洲哲学”、“现代欧陆哲学”、“西方伦理学史”、“西方伦理思想专题”、“价值伦理学”等。


科研情况：                                                       

一、研究领域
伦理学、近现代哲学、价值现象学

 

   二、主要学术活动

  1. 2004年、2016年和2018年分别参加第十届、第二十一届和第二十三届中国现象学年会提交论文并发言；

  2. 2006年暑假参加由中国人民大学主办的中英美暑期哲学学院研讨班（主题：康德伦理学）；

  3. 2008年暑假参加香港中文大学现象学大师班（主题：胡塞尔和梅洛-庞蒂的意向性）；

  4. 2010年10月参加台湾大学哲学系“当代欧陆哲学的价值与存有论”国际学术会议，提交并宣读论文“价值认定与价值存在——Max Scheler 的价值现象学探析”；

  5. 2017年参加第十四届舍勒国际会议（主题：“均衡时代中的舍勒与亚洲思想”，广州），提交论文并发言；

  6. 2016年10月和2017年6月分别组织南开大学伦理学论坛两次中青年学者学术讨论会，会议主题分别是“价值论与现象学”和“情感与价值”。

 

三、主持项目

“现象学运动中的情感与价值问题研究”（2013年国家社科基金青年项目）（已结项）；

“价值哲学与意向性问题研究”（2011年教育部青年基金项目）（已结项）；

“马克斯·舍勒的价值伦理学研究”（2008年教育部后期资助项目）（已结项）。
