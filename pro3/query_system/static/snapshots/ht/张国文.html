张国文（1983.6- ），男，安徽合肥人。南开大学历史学院考古学与博物馆学系副教授，南开大学文博考古市级实验教学示范中心科技考古实验室负责人，加拿大西蒙弗雷泽大学考古学系访问学者（2018年12月-2019年12月）。研究方向为北方民族考古、科技考古（生物考古）。博士毕业于中国科学院研究生院科技史与科技考古系，获理学博士学位，期间于2010年9月至2011年9月前往德国马普研究所莱比锡人类进化研究所博士联合培养。

北方民族考古、科技考古（生物考古）

三、田野考古实践经历

2004年10－12月：安徽省考古研究所和安徽大学主持的安徽省芜湖市繁昌县瓜墩遗址考古发掘；

2007年8－9月：河南省南阳市桐柏县汉代矿冶遗址调查与试掘；

2007年9－11月、2009年3-5月：国家南水北调项目工程——河南省淅川县滔河乡申明铺遗址调查发掘；

2009年11月：国家南水北调项目工程——河北省元氏县赞皇遗址发掘与整理；

2012年4月-7月、2012年10月-12月：湖北郧县龙门堂墓群田野考古发掘与资料整理。
