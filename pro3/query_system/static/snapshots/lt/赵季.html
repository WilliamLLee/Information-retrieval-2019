赵季

中国语言文学系

教授、博士生导师

Tel.:0086-22-23503845转8206 fax:086-022-23508247

Zhaoji@nankai.edu.cn

个人简介

赵季，1951年5月3日生，男，山东惠民人。1983年7月南开大学中文系本科毕业，获文学学士学位。1986年7月南开大学中文系研究生毕业，获文学硕士学位。1986年9月任教于南开大学中文系，现为南开大学中文系教授、博导。从事中国古代文学研究及韩国汉诗、域外诗话研究。

 

主要著作： 

《新罗高丽朝鲜汉诗集成》（第二辑，57册，第一作者），凤凰出版社2018年5月。

《韩国日本吟诵文献辑释》（第一作者），天津教育出版社2017年12月。

《新罗高丽朝鲜汉诗集成》（第一辑，70册），凤凰出版社2017年5月。

《中朝三千年诗歌交流考论》（第一作者），南开大学出版社2016年12月。

《诗话丛林校注》（第二作者），人民文学出版社2015年。 

《明洪武至正德中朝诗歌交流系年》（第一作者），人民文学出版社2014年。 

《韩国文人名字号训诂辞典》（第三作者），韩国以会社2014年。 

《箕雅五百诗人本事辑考》（上、下，第一作者），人民文学出版社2013年。 

《足本皇华集》（上、中、下），凤凰出版社2013年。 

《韩国诗话全编校注》（12册，主编），人民文学出版社2012年。 

《韩国汉文学史》（译著，第一译者）， 凤凰出版社2012年。

《韩国诗话人物批评集》（5册，第三作者），韩国宝库社2012年。 

《箕雅研究》（第一作者），南开大学出版社2010年。 

《箕雅校注》（上、下），中华书局2008年。
