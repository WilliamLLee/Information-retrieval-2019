发布者：sudy发布时间：2019-10-29浏览次数：10

陈鹏

传播学系

系主任，研究生导师


chenpeng@nankai.edu.cn

个人简介

陈鹏，博士，新闻传播学博士后、理论经济学博士后，传播学系新闻学教研室副教授。研究方向为：传播理论与方法、传媒经济制度研究、新媒体产业与舆情分析、影视内容管理等。曾赴国内外多个地区进行媒介调研，参与多项课题研究，发表研究报告、论文数十篇，参与编著、翻译著作多部。曾讲授《传播学》、《传播统计学原理》、《电视节目管理》等课程。


【学习经历】

北京广播学院学士；

中国传媒大学硕士；

中国传媒大学博士；

清华大学与山东广电联合博士后；

南开大学博士后；

“教育部·电通广告人才培养基金项目”电通高级人才研究项目研修员，赴日本研修广告与媒体运营。


【科研活动】

主持：

国家社科基金项目“三网融合背景下的广播电视制播分离与市场适应机制研究”

天津市社科基金项目“广播电视制播体制改革研究”

山东省创新基金项目“全媒体语境下的山东广播影视发展战略研究”

南开大学青年项目“中国媒介制度变革研究”


参与：

近年来，先后参与多项科研课题的研究，主要包括：

科研课题“电视频道专业化与盈利模式研究”；

科研课题“传媒集团化运营与管理研究”；

科研课题“中国传播学流派研究”；

科研课题“现代体育中的组织传播”；

科研项目“陕西电视台频道管理与品牌建设研究”（频道规划部分）；

科研项目“湖北影视频道产业化经营执行方案研究”；

科研项目“电视台管理研究”（宣传管理部分）；

科研项目“中国证券市场投资者信心指数抽样方案研究”；

科研项目“国民数字生活指数模型研究”；

科研项目“奥运背景下的北京文化创意产业发展研究”（北京市项目）；

科研项目“中国网络影响力研究”（中国广电协会项目）；

资助项目“中国媒介30年制度变革研究”（北京市软科学资助项目）。


【科研成果】

在核心期刊和专业刊物上发表《破冰前行的电视产业变革：制播分离与节目公司化运作趋势分析》、《我国农村广播电视覆盖的优化实现》、《公信力战略与整合营销传播——打造电视媒介核心竞争力》、《世界杯经济：魅力足球的全球经济影响力》、《股市：繁荣之喜与泡沫之忧》、《新媒体内容产业的数据库市场竞争分析》、《传媒经济再腾飞的历史契机》等论文。

参与编著《媒介经营与管理总论》、《传媒集团运营机制》、《广播电视经营与管理模式》、《世界五大媒介集团经营之道》、《应用传播学史纲》、《体育传播》、《中国电视媒体运营管理实务》、《中国电视网络影响力报告》、《制度与空间——中国媒介制度变革论》等著作。

翻译完成《传媒经济学：市场、产业与观念》。


【教学活动】

校级精品课《传播学概论》主讲教师之一；

主讲课程：

本科生：《传播学概论》，《传播学研究方法》、《传媒经济学》、《消费社会与大众传播》、《专业英语》；

研究生：《新闻学理论》、《媒体经济研究》、《消费社会与文化产业》、《传播学理论》。


【科研方向】

传播理论与方法、传媒经济制度研究、新媒体产业与舆情分析、影视内容管理。
