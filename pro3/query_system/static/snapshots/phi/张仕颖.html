个人简历：

张仕颖，男，1973年生，籍贯贵州遵义。2003年在武汉大学哲学系获得外国哲学博士学位，师从著名学者赵林教授，现任南开大学哲学院宗教学教研室主任。主要学术研究领域为宗教哲学、中世纪哲学、基督教思想史和西方宗教学理论。曾访学于香港中文大学（2005年）、日本早稻田大学（2006年）、台湾辅仁大学（2007年）、美国耶鲁大学（2013年），香港道风山汉语基督教文化研究所（2016年），多次参加国际学术会议，2017年8月受邀参加在德国维滕堡举行的第13届世界路德研究大会。

学术兼职：

   中国宗教学会理事

   香港道风山汉语基督教文化研究所研究员

   香港中文大学基督教与中国社会研究中心特约研究员

   加拿大不列颠哥伦比亚大学附属维真神学院（Regent College）南开大学合作项目执行人

     香港道风汉语基督教文化研究所南开大学合作项目执行人
