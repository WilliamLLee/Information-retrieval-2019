个人简历：

1978年2月入河南大学数学系读本科，后获理学学士学位，1989年7月获中国科学院软件研究所理学硕士学位。1983年起在河南大学政治系工作，1994年—2000年任河南大学政治系副教授，2000年任教授。2002年1月调入南开大学哲学系工作。一直从事现代逻辑的教学与研究。
