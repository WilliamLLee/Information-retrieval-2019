个性化感言                                                             
许多没有出路的逻辑学研究，只是因为对哲学家昔日的失败一无所知才得以维持。可见哲学对于逻辑发展具有极端的重要性。如果说哲学打开了人类心灵之窗，那么可以说逻辑是哲学的精髓。
