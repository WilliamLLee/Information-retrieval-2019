个人简历：

1974年10月生，山西省昔阳人，2004年结束学生生涯来南开大学任教。

1993年至2000年求学于吉林大学，先后获得哲学学士和哲学硕士学位（师从姚大志教授）；
2000年至2004年求学于北京大学，师从杜小真教授获得博士学位；
2002年9月至2003年8月在法国巴黎-索邦大学学习；
2010年9月至2011年4月在法国雷恩第一大学访学
