个人简历：

李淑梅, 女，祖籍河北黄骅，天津塘沽人，1954年6月生。1973-1976年，南开大学哲学系上学， 1980-1983年，南开大学哲学系硕士研究生，获哲学硕士学位，1986-1989年,南开大学哲学系博士研究生，获哲学博士学位。1994年一1995年在希腊雅典大学哲学系读博士后，2002年在美国哈佛哲学系访问学习半年。南开大学哲学系教授，博士生导师。
