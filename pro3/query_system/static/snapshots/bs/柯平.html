获奖成果

科研获奖：
著作《图书馆知识管理研究》（北京图书馆出版社2006）2009年获国家教育部普通高等学校人文社会科学研究成果著作二等奖；
著作《书目情报系统理论研究》（书目文献出版社1996）1998年获国家教育部普通高等学校第二届人文社会科学研究成果著作三等奖；
著作《文献经济学》（中国书籍出版社2001）获天津市第八届社会科学优秀成果奖，中国社会科学情报学会第三届优秀科研成果一等奖；
著作《信息管理概论》（科学出版社2002）获天津市第九届社会科学优秀成果奖；
论文《公共文化服务体系中的图书馆与社会合作实证研究》获天津市第十二届社会科学优秀成果奖。

教学获奖：
1996年主持教学项目《实施图书馆学专业“知识、能力、素质”综合培养的实践教学模式》通过省级鉴定并获河南省教学成果省级一等奖
2002年《书目情报需求与服务组织》获教育部全国普通高等学校优秀教材一等奖（国家级奖）；2007年《关于图书馆学专业优质教学资源建设》获南开大学“本科教育教学改革与创新”论文二等奖。
