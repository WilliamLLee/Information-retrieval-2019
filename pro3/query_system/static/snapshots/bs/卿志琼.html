教育背景

1983-1987年，兰州大学哲学系，获哲学学士；

1995-1998年，陕西财经学院经济系，获商业经济学硕士；

2000-2003年，南开大学经济系，获经济学博士。



工作经历

1987年6月-1999年，陕西财经学院教师;

2000年-2003年6月，西安交通大学教师 ;

2003年7月，南开大学商学院教师，副教授.



获奖成果

1、《中国十位著名经济学家批判》，获天津市第九届社科成果三等奖。

1、“从认知科学到经济学：情绪介入经济决策的内在机理研究”，《财经研究》2012年第1期。

3、“同情介入决策的实验研究——基于收废品老伯的实验”，《南开管理评论》2010年第4期。

4、“情绪介入经济决策研究的演变路径——从古典经济学到神经经济学”，《财经研究》2009年第9 期，《新华文摘》2009年第23期转载，《高等学校文科学术文摘》2009年第6期转载。

5、《有限理性、心智成本与经济秩序》，经济科学出版社，2006年。

6、认知偏差与理性选择——基于“最后通牒博弈”实验的认知博弈，《南开经济研究》2005年第1期。人大复印资料《理论经济学》2005.6转载 。

7、“循规与违规：中国市场制度变迁的另一种解释”，《经济评论》2004年第1期。
