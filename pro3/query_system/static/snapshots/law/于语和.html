于语和

发稿时间：2017-06-23浏览次数：4849

  

  

于语和 教授

  

研究领域：主要研究方向：中国法律史（制度史，法律思想史）、中国民间法、西方法律思想史、中外法律文化比较研究、中西政治文化比较研究。

  

个人简介

姓名：于语和

民族：汉族

籍贯：天津宁河

出生年月：1962年9月

政治面目：中共党员

学历：博士研究生毕业；博士后出站

学位：法学博士；中国人民大学法学博士后

单位：南开大学法学院

职称：教授，博士生导师

职务：原天津大学法学系主任，现任南开大学法学院法史法理教研室主任；2004-2005年，受中组部和团中央委派，赴宁夏司法厅挂职，任司法厅党委委员、厅长助理（副厅级），分管基层处、法宣处和自治区依法治区领导小组办公室。因工作认真，成绩突出，2005年9月，受到中组部表彰。2010年4月至今，在天津市人大代表进修学校挂职，天津市人大常委会党组任命本人为人大代表进修学校副校长，分管教务和基建、绿化及设备维修工作。

学术兼职：中国法学会法学教育研究会理事；全国外国法制史研究会理事；天津法学会理事、专家评审委员会委员；天津法学会法理学分会副会长；天津政治学会理事；天津市“四五”、“五五”和“六五”普法讲师团成员。

  

获奖情况：

2002年，天津市第八届社科优秀成果评奖获论文三等奖。

2002年，获第三届“全国杰出中青年法学家”提名奖。

2003年，获天津市第二届依法治市理论研究征文优秀论文一等奖。

2004年，获天津市教育系统优秀思想政治工作者称号。

2004年，被评为天津市首届十大杰出中青年法学家。

2005年，获南开大学社会科学研究优秀成果奖。

2006年，被评为全国四五法制宣传教育先进个人。

2006年，天津市第十届社科优秀成果著作类二等奖（成员之一）。
